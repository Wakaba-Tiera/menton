## 1. 기본 개념

### 1.1 레지스터

멘똔랭은 **다수의 정수 레지스터**를 가지며,  
항상 **현재 선택된 레지스터 하나**를 대상으로 연산이 수행된다.

레지스터 이름만 단독으로 한 줄에 쓰면,  
그 레지스터가 **현재 레지스터**로 선택된다.

```text
멘똔
배털
```

---

#### 기본 레지스터

기본 레지스터는 친목방 멤버들의 이름을 사용한다.  
총 **9개**가 있으며, 프로그램 시작 시 기본 선택 레지스터는 `멘똔`이다.

```text
멘똔
배털
정빵
애플리프트
깨무이
혁두
턱살개구리
잉진이
민짜이
```

이 레지스터들은 다음과 같은 용도로 자유롭게 사용할 수 있다.

- 변수 역할
- 계산 중간값 저장
- 반복문 카운터
- 곱셈용 보조 레지스터

---

#### 확장 레지스터

기본 레지스터 외에도, 멘똔랭에는 **확장 레지스터**가 존재한다.

확장 레지스터는 다음 규칙으로 생성된다.

```text
<기본 음절>가<기본 음절>가
```

사용 가능한 기본 음절은 다음 7가지이다.

```text
멘, 빵, 깨, 털, 두, 덜, 애
```

이 조합으로 만들어지는 모든 레지스터는 **자동으로 존재**하며,  
총 **49개**의 확장 레지스터가 제공된다.

예시:
```text
멘가멘가
멘가빵가
깨가털가
애가두가
```

확장 레지스터는 기본 레지스터와 **완전히 동일한 방식**으로 동작하며,

- 값 저장
- 연산 대상
- 곱셈 피연산자
- 출력 대상

등 모든 용도로 사용할 수 있다.

## 2. 숫자 표기 (웃음 숫자)

멘똔랭에서 모든 숫자는 **웃음 숫자 방식**으로 표현된다.  
이 방식은 각 자릿수를 특정한 웃음 토큰으로 표현하고,  
토큰의 반복 횟수로 값을 나타내는 **자리수 기반 표기법**이다.

---

### 2.1 자리 토큰

각 자릿수는 다음 토큰에 대응된다.

| 자리 | 토큰 |
|---|---|
| 1의 자리 | 훠 |
| 10의 자리 | 훳 |
| 100의 자리 | 허 |
| 1000의 자리 | 헛 |

- 자릿수는 **10진수 기준**이다.
- 10,000의 자리는 다시 `훠`부터 순환한다.
  - (10,000 → 훠, 100,000 → 훳, …)

---

### 2.2 기본 규칙

- 같은 자리 토큰을 **반복한 횟수 = 해당 자리의 값**
- 숫자는 **큰 자리 → 작은 자리 순서**로 작성한다.
- 값이 0인 자리는 **아무 것도 쓰지 않는다**.

예시:
| 값 | 표현 |
|---|---|
| 1 | 훠 |
| 2 | 훠훠 |
| 5 | 훠훠훠훠훠 |
| 10 | 훳 |
| 20 | 훳훳 |
| 101 | 허 훠 |
| 111 | 허 훳 훠 |

---

### 2.3 6 이상 표현 (`훠러`)

한 자리에서 값이 **6~9**일 경우,  
`훠러` 접두사를 사용한다.

규칙:
- `훠러` + (자리 토큰을 **값 − 5** 만큼 반복)

예시:
| 값 | 표현 |
|---|---|
| 6 | 훠러훠 |
| 7 | 훠러훠훠 |
| 8 | 훠러훠훠훠 |
| 9 | 훠러훠훠훠훠 |

10의 자리 예시:
| 값 | 표현 |
|---|---|
| 60 | 훠러훳 |
| 90 | 훠러훳훳훳훳 |

---

### 2.4 `찢` : 0으로만 이루어진 네 자리 묶음

`찢`은 **연속된 네 자리(0000)** 를 의미하는 특수 토큰이다.

- `찢` 하나는 **10,000 단위 이동**을 의미한다.
- 실제 값은 없으며, **자리만 건너뛴다**.

예시:
| 값 | 표현 |
|---|---|
| 10,000 | 찢 훠 |
| 20,000 | 찢 훠훠 |
| 10,001 | 찢 훠 훠 |
| 12,345 | 찢 허허 허허훳훳훳훳 훠훠훠훠훠 |

- `찢`은 반드시 **자리 경계(4자리 단위)** 에서만 사용된다.
- 네 자리가 전부 0일 경우, 해당 자리는 반드시 `찢`으로 표현해야 한다.

---

### 2.5 음수

음수는 숫자 앞에 `뭐꼬`를 붙여 표현한다.

```text
뭐꼬 훠훠
```

위 예시는 `-2`를 의미한다.

---

### 2.6 요약

- 웃음 숫자는 **10진수 자리 기반 표기**
- 토큰 반복 = 값
- 0은 생략, 네 자리 0은 `찢`
- 6 이상은 `훠러` 사용
- 음수는 `뭐꼬` 접두사 사용


## 3. 값 조작 명령어

### 3.1 값 설정

```text
하요하요 <숫자>
```

- 현재 레지스터 값을 해당 숫자로 설정
- 숫자를 생략하면 0

```text
하요하요 훠훠훠
```

---

### 3.2 값 초기화

```text
바요바요
```

- 현재 레지스터를 0으로 초기화

---

### 3.3 덧셈 / 뺄셈

```text
누이 좋고 <숫자>
매부 좋고 <숫자>
```

- 숫자를 생략하면 기본값은 1

---

### 3.4 곱셈

```text
아주 좋고 <피연산자>
```

- 현재 레지스터 값에 곱한다.
- **레지스터 또는 숫자 모두 사용 가능**
