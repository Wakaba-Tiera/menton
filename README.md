# MentonLang

멘똔랭은 **레지스터 기반 난해(Esoteric) 프로그래밍 언어**로, 모든 키워드와 숫자 표기가 독특한 한국어 토큰으로 이루어져 있다.

👉 **웹에서 바로 실행하고 테스트할 수 있다**  
https://wakaba-tiera.github.io/menton/web/

별도의 설치 없이 문법을 시험해보고 출력 결과를 즉시 확인할 수 있다.

---

## 1. 기본 개념

### 1.1 레지스터

- 프로그램에는 여러 개의 레지스터가 존재한다.
- 항상 **현재 선택된 레지스터 하나**만 연산 대상이 된다.
- 레지스터 이름만 단독으로 한 줄에 쓰면 현재 레지스터가 변경된다.

예시:
```text
멘똔
배털
```

---

### 1.2 주석

- `#` 이후의 내용은 모두 주석으로 처리되어 무시된다.

```text
하요하요 훠훠  # 이 부분은 실행되지 않는다
```

---

## 2. 숫자 표기 (웃음 숫자)

멘똔랭에서 숫자는 **웃음 숫자 방식**으로 표현한다.

### 2.1 자리 토큰

| 자리 | 토큰 |
|---|---|
| 1의 자리 | 훠 |
| 10의 자리 | 훳 |
| 100의 자리 | 허 |
| 1000의 자리 | 헛 |

- 같은 토큰을 **반복한 횟수 = 해당 자리의 값**
- 6~9는 `훠러` 접두사를 사용한다.

---

### 2.2 예시

| 값 | 표현 |
|---|---|
| 1 | 훠 |
| 3 | 훠훠훠 |
| 5 | 훠훠훠훠훠 |
| 6 | 훠러훠 |
| 9 | 훠러훠훠훠훠 |
| 10 | 훳 |
| 42 | 훳훳훳훳 허허 |

---

### 2.3 음수

- 음수는 앞에 `뭐꼬`를 붙인다.

```text
뭐꼬 훠훠
```

---

## 3. 값 조작 명령어

### 3.1 값 설정

```text
하요하요 <숫자>
```

- 현재 레지스터 값을 해당 숫자로 설정
- 숫자를 생략하면 0

```text
하요하요 훠훠훠
```

---

### 3.2 값 초기화

```text
바요바요
```

- 현재 레지스터를 0으로 초기화

---

### 3.3 덧셈 / 뺄셈

```text
누이 좋고 <숫자>
매부 좋고 <숫자>
```

- 숫자를 생략하면 기본값은 1

```text
누이 좋고 훠훠
매부 좋고 훠
```

---

### 3.4 곱셈

```text
아주 좋고 <피연산자>
```

- 현재 레지스터 값에 곱한다.
- **레지스터 또는 숫자 모두 사용 가능**

레지스터 사용:
```text
아주 좋고 배털
```

숫자 직접 사용:
```text
아주 좋고 훠훠훠
```

---

## 4. 조건문

### 4.1 IF

```text
건방진 <숫자> [비교자]
```

비교자:
- `응나멘똔` : 크다 (`>`)
- `응너도혁` : 작다 (`<`)
- 생략 시 같음 (`==`)

```text
건방진 훠훠 응나멘똔
```

---

### 4.2 ELSE

```text
정신이 나갔어 정신이
```

---

### 4.3 블록 종료

```text
쉐끼마
```

---

## 5. 반복문 (WHILE)

```text
좋다좋다 <숫자> [비교자]
    ...
쉐끼마
```

- 조건이 참인 동안 반복
- 조건 형식은 IF와 동일

---

## 6. 출력

### 6.1 출력 블록 시작

```text
와타시는
```

---

### 6.2 숫자 출력 종료

```text
이라는 것이야
```

- 숫자 또는 레지스터 값을 그대로 출력

---

### 6.3 문자 출력 종료

```text
한다는 것이야
```

- 숫자를 ASCII 코드로 해석해 문자 출력

---

### 6.4 출력 전용 토큰

| 토큰 | 의미 |
|---|---|
| `~` | 공백 |
| `ㅢ?!` | 줄바꿈 |

---

```text
와타시는
```

```text
이라는 것이야
```

출력:
```
6
```

---

### 예제 3: 반복 출력

```text

좋다좋다 훠
와타시는

매부 좋고 훠
쉐끼마
```

출력:
```
5 4 3 2 1 
```

## 8. 로컬 실행 방법

```bash
python mentonlang_v3.py program.txt
```

- `으이?` 토큰은 줄바꿈으로 자동 변환된다.
- 실행 결과는 표준 출력으로 출력된다.
