# Menton Language (멘똔) 문법 설명서

이 문서는 **멘똔(Menton)** 언어의 문법을 실제 예제와 함께 설명한다.  
멘똔은 레지스터 기반, 줄 단위 명령형 언어이며 독특한 **웃음 숫자 표기법**을 지원한다.

---

## 1. 기본 개념

### 1.1 프로그램 구조
- 프로그램은 **줄 단위**로 실행된다.
- 빈 줄과 `#`으로 시작하거나 포함된 주석은 무시된다.
- 모든 연산은 **현재 선택된 레지스터**를 기준으로 수행된다.

---

## 2. 레지스터

### 2.1 레지스터 토큰
- 기본 토큰: `멘, 빵, 깨, 털, 두, 덜, 애`
- 레지스터는 항상 다음 형태를 가진다:

```
<기본토큰>가<기본토큰>가
```

예:
```
멘가멘가
빵가깨가
애가두가
```

- 총 **7 × 7 = 49개**의 레지스터가 존재한다.
- 레지스터 이름이 단독으로 한 줄에 나오면 **현재 레지스터 선택**을 의미한다.

---

## 3. 숫자 리터럴

멘똔은 두 가지 숫자 표현을 지원한다.

### 3.1 아랍 숫자
일반적인 정수 표기:
```
123
-45
```

---

### 3.2 웃음 숫자 표기법

#### 3.2.1 자리 토큰 (4개 순환)

| 자리 | 토큰 |
|---|---|
| 1의 자리 | 훠 |
| 10의 자리 | 훳 |
| 100의 자리 | 허 |
| 1000의 자리 | 헛 |

- 10,000부터는 다시 `훠 → 훳 → 허 → 헛` 순으로 **순환**한다.

---

#### 3.2.2 보조 토큰

- `훠러` : 6~9를 표현할 때 사용하는 **+5 접두사**
- `찢` : **연속된 4자리(0000)** 전체가 0일 때 사용

---

#### 3.2.3 한 자리 숫자(0~9) 표현

| 숫자 | 표현 방식 |
|---|---|
| 0 | 생략 |
| 1~4 | 자리 토큰 반복 |
| 5 | 자리 토큰 5번 반복 |
| 6~9 | `훠러` + 자리 토큰 (값-5)번 |

예 (10의 자리):
```
3  → 훳훳훳
7  → 훠러훳훳
```

---

#### 3.2.4 여러 자리 숫자

- **큰 자리 → 작은 자리 순서로 이어붙임**

예:

- 423
```
허허허허훳훳훠훠훠
```

- 30000
```
훠훠훠찢
```

- 1000070000
```
훳훠러훠훠찢
```

---

## 4. 값 설정 및 연산

### 4.1 값 설정

```
하요하요 10      # 현재 레지스터 = 10
하요하요 훳훳    # 웃음 숫자 사용
하요하요         # 0으로 설정
```

### 4.2 리셋
```
바요바요          # 현재 레지스터 = 0
```

### 4.3 덧셈 / 뺄셈

```
누이 좋고 3      # +3
매부 좋고 1      # -1
```

숫자를 생략하면 기본값은 1이다.

---

### 4.4 곱셈

```
아주 좋고 빵가빵가
```

- 현재 레지스터 × 지정한 레지스터 값

---

## 5. 조건문 (IF / ELSE)

### 5.1 IF

```
건방진 10
  ...
쉐끼마
```

의미: 현재 레지스터 == 10

### 5.2 비교 연산자

- `응나멘똔` : >
- `응너도혁` : <

예:
```
건방진 5 응나멘똔   # > 5
```

### 5.3 ELSE

```
건방진 0
  ...
정신이 나갔어 정신이
  ...
쉐끼마
```

---

## 6. 반복문 (WHILE)

```
좋다좋다 10 응너도혁
  ...
쉐끼마
```

의미: 현재 레지스터 < 10 동안 반복

---

## 7. 출력

### 7.1 출력 블록 시작

```
와타시는
```

---

### 7.2 숫자 출력

```
와타시는
멘가멘가
10
이에요
```

- 각 항목을 숫자로 출력 + 줄바꿈

---

### 7.3 ASCII 문자 출력

```
와타시는
72
101
108
108
111
한다는 것이야
```

→ `Hello`

---

## 8. 주석

- `#` 이후의 모든 내용은 무시된다.

```
하요하요 10  # 초기값 설정
```

- 모든 연산은 현재 레지스터 기준

- `찢`으로 0000 그룹 압축
- 단순하지만 표현력 있는 밈 언어
